# Scrolling Portfolio Site

A design-first portfolio project for a travelling software engineer and site reliability engineer.

## Status

Planning and design phase. No frontend framework has been scaffolded yet.

## Docs

- [Initial Design Brief](./docs/initial-design-brief.md)
- [Technical Direction](./docs/technical-direction.md)

## Development

This project uses Vite, React, TypeScript, GSAP, and Lenis.

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
